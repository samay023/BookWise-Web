# BookWise-Web
A booking website that helps anyone to manage booking easily

# Tech Stack (ServerSide)
```
Apollo Server
GraphQL
MongoDB
```
# Tech Stack (ClientSide)
```
React
Redux
Redux-thunk
Apollo Client
Bootstrap
```

## Getting Started

Clone or download the zip package from github

### Prerequisites

```
NodeJS
```

### Installing

```
Goto root directory
Run npm install
Run npm run dev
